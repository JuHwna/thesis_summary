# 1. 터미널이란?
- 컴퓨터와 텍스트로 대화할 수 있는 창
- ls
  - mode 의 앞글자 d는 디렉토리
  - -a 면 파일

## 2. 줄바꿈
- \하고 엔터하면 줄바꿈함
- 윈도우 ctrl+enter or alt_enter

## 3. terminal setup
- /terminal-setup : 터미널 셋업 명령어를 설정할 수 있음
- 이걸로 줄바꿈 설정할 수 있음
  - shift+enter 단축키가 설정된거를 확인할 수 있음

## 4. 프로젝트 초기화(/init) 및 기본 사용법
- 프로젝트 시작할 때 /init 실행한다고 보면 됨
- /init 실행 시
  - 클로드 코드가 현재 디렉토리를 읽고
  - bash 명령어를 통해서 현재 디렉토리에 무엇이 있는지 확인함
  - claude코드가 claude.md 파일을 생성하는지 물어봄
    - 쓰기 작업을 물어보는 것
  - 작업 완료됨

- 프로젝트가 next.js 기반 프로젝트였다면
  - 클로드.md 파일에 프로젝트에 대한 기술 스택, 어떻게 개발되었는지 가이드라인이 작성되었을 것임.
  - 클로드 코드가 개발을 할 때 claude.md파일을 확인해서 일관성 있게 개발할 수 있게 도와줌
  - claude.md => 메모리, 기억하는 역할을 수행함
    - 프로젝트 구조, 디렉토리 구조가 어떠한지
    - 가이드라인을 확인해서 일관성 있게 개발할 수 있도록 도와줌

- 예시)
~~~
/init 프로젝트 초기화를 한국어로 진행해주세요.
## 언어 및 커뮤니케이션 규칙
- **기본 응답 언어**: 한국어

- **커밋 메시지** : 한국어로 작성
- **문서화** : 한국어로 작성
- **변수명/함수명** : 영어(코드 표준 준수) 
~~~

- esc를 두번 누르면 이전에 내가 진행했던 프롬프트 리스트를 보여주고 그거를 요청할 수 있게 됨
  - 여기서 esc 두 번 누른 뒤에 그 리스트를 선택해서 진행시키면 일부 프롬프트 리스트가 사라짐
