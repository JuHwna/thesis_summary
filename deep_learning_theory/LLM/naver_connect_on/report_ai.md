# 완성도를 높여가는 기획서 작성법
- AI는 초안을 만드는 도구일뿐, 완성은 사람이 해야 한다.
  - 검증 없이 사용하면 이런 문제 발생함

## 강의 목표
- AI로 기획 서 초안 체계적으로 작성
- AI 결과물의 오류를 찾아내는 검증 기법 활용
- 좋은 기획서의 구조를 이해하고 적용
- 실전에서 바로 쓸 수 있는 프롬프트

## 좋은 기획서의 기본 구조(5가지 구조)
1. 배경 및 문제 인식(Why) 왜 이 기획이 필요한가?
   - 현재 어떤 문제가 있는가?

2. 목표 설정(What) 이 기획을 통해 무엇을 달성할 것인가?
   - 구체적이고 측정 가능한 목표는?
3. 기획 내용 및 방법 (How) 어떻게 실행할 것인가?
   - 일정, 인력, 예산은?

4. 기대 효과 (So What)
   - 이 기획이 성공하면 어떤 결과가 나오는가?

5. 결론 및 요청사항
   - 핵심 내용 요약과 필요한 협조 사항은?
  
### 올바르게 입력하는 방법
- 단계별로 작성해야 함
~~~
중소기업(직원 50명 규모)를 위한 고객 응대 AI 챗봇 도입 기획서를 작성하려고 해.
아래 구조로 목차를 만들어줘:
1.배경 및 문제인식(Why)
2.목표 설정(What)
3.기획 내용 및 방법(How)
4.기대효과(So What)
5.결론 및 요청사항

각 세션에 어떤 내용이 들어가야 할지 간단히 설명도 붙여줘.
~~~

- AI가 작성해 준 기획서는 내 의도를 정확하게 파악하지 못함(핵심)
  - 배경 및 문제인식을 섹션에다가 내가 어느 정도 정보를 직접 입력해줘야 함(핵심)

- 5단계로 나누어서 집중 시키기
  - AI는 한 번에 굉장히 많은 일을 시키는 것보다 단계별로 쪼개서 해당 섹션 내용을 좀 더 집중하게 만들어주면 더 좋은 내용으로 만들어냄

- genspark
  - 플러스 버튼 누르면 팩트체크를 클릭 후, 작성한 워드파일 업로드 후
    - "팩트체크 시작" <- 이렇게만 입력하면 팩트체크를 해줌
    - 결과값이 gpt랑 다른 이유 : genspark는 실제로 계획을 세우고 에이전트가 리서치를 해준 다음에 그 다음에 실제로 스크린 캡쳐를 해서 그 사람에 대해서 검증을 해나감

- genspark가 알려준 걸로 수정한 뒤에 논리가 이상한 경우가 나올 수 있음
  - 마지막으로 gpt한테 내가 작성한 보고서와 기획서에 대해서 논리 검증을 한 번 더 시키면 논리적인 기획서 작성 가능

## 핵심 정리
1. AI는 도구다
   - AI가 기획서를 써주지만 완성은 사람이 해야 함
   - 초안 작성에는 AI를 검증과 완성에는 사람의 판단을 사용
2. 단계별로 작성하라
   - 한번에 전체를 쓰지 말고 섹션별로 나눠서 작성
   - 구조 -> 배경 -> 목표 -> 방법 -> 효과 -> 결론 순서로 진행하면 됨
3. 반드시 검증
   - 팩트체크 : 숫자, 통계, 인용, 고유명사
   - 논리 검증 : 인과관계, 근거, 누락 내용
   - 구조 점검 : 가독성, 흐름, 분량

# 기획서에 활용할 이미지 시각 자료 만들기

## 강의 목표
- 구글 나노바나나로 원하는 이미지를 정확히 생성할 수 있음
- 효과적인 이미지 프롬프트 작성법을 터득
- 냅킨 AI로 복잡한 개념을 자동으로 시각화할 수 있음
- 실무에 바로 적용 가능한 비주얼을 직접 제작

## 나노 바나나 3가지 핵심 강점
1. 캐릭터 일관성 유지
   - 기존 AI는 같은 캐릭터로 여러 이미지를 만들 때마다 얼굴이나 스타일이 바뀌는 한계가 있었지만 나노바나나는 여러 편집 단계에서도 인물의 얼굴, 포즈, 배경을 동일하게 유지

2. 자연어 편집 이해력
   - 마스크 작업 없이 텍스트 명령만으로 특정 부분을 수정하거나 교체할 수 있음
   - 별도의 프롬프트를 배울 필요 없이 자연어로 입력하면 됨
3. 빠른 생성 속도
   - 나노바나나는 최소한의 지연 시간으로 이미지를 생성하며 현재 선도 모델보다 최대 8배 빠르게 작동함
   - 미드저니는 1분, DALL-E는 30초 걸릴 때 나노바나나는 5~10초면 끝
  
- 구글 제미나이 앱이나 구글 AI 스튜디오에서 사용 가능함
  - 제미나이 앱은 생성 이미지 적음
  - 구글 AI스튜디오는 많이 생성 가능
 
## 프롬프트 작성 팁 : 좋은 프롬프트의 3가지 원칙
1. 구체적으로 말해라
2. 스타일을 명시하라
   - 사실적인 느낌을 원한다면(아래 예시)
     - 포토리얼스틱
     - 실사 사진 같은
     - DSLR 카메라로 촬영한 듯한
     - 스튜디오 조명
   - 일러스트 느낌을 원한다면:
     - 플랫 디자인 일러스트
     - 수채화 스타일
     - 파스텔 톤 일러스트
     - 미니멀리즘
   - 3D 느낌을 원한다면:
     - 3D 렌더링
     - 픽사 스타일 3D 애니메이션
     - 클레이 모형 같
3. 원하지 않은 것도 말하라(은근히 중요)
   - 사실적인 여성 초상화, 자연광, 노이즈 없이, 왜곡 없이, 흐릿하지 않게
   - "~하지 않게"를 추가하면 품질이 확 올라감.
  
## 프롬프트 작성이 어려울 때 : AI에게 도움받기
- 프롬프트
  - 나노바나나 AI 이미지 생성 모델에 대해 검색해줘. 특히 효과적인 프롬프트 작성 방법을 찾아줘.
  - 첫번째 맥락을 쌓는 것
    - 그냥 바로 '나 지금 현재 제미나이 나노바나나 이미지 프롬프트 만들어줘'하면
      - 할루시네이션이 작동함
    - 그래서 검색을 통해서 넣어줘야 함

- 그 다음 프롬프트
~~~
나는 패션 브랜드 SNS에 올릴 Y2K 감성의 GenZ 모델 이미지를 만들고 싶어.
타겟 : 18~25세 Z세대
스타일: Y2K 감성, 2000년대 초반 복고용
느낌 : 밝고 생기있는, 레트로 감성

나노 바나나에 입력할 프롬프트 5개를 만들어줘. 각각 조금씩 다른 스타일로
~~~

- 구글 스튜디오 : aistudio.google.com 입력
  - 왼쪽에 chat 혹은 try nanobanna 클릭
  - 하단에 프롬프트 창에 입력
 
- 그 다음 프롬프트
  - 동양인 한국인을 원해
 
- 그 다음 프롬프트
  - 이미지 복사를 해서 업로드
  - 첨부한 이미지들과 비슷한 느낌의 Y2K 감성 GenZ 모델 이미지를 만들고 싶어. 프롬프트를 써줘.

## Gemini 2.5 Flash 이미지 합성 팁
- 원칙 1 : 유지할 것을 명확히 말하라
  - 얼굴과 헤어스타일은 그대로 유지, 제품 색상과 디자인은 정확히 유지
  - "바꾸지 말아야할 것"을 확실히 못 박아야 원하는 부분만 바뀐다.
- 원칙 2 : 바꿀 것을 구체적으로 명시하라
  - 배경을 회색 그라데이션 스튜디오 배경으로 의상을 네이비 블루 정장으로 "바꿀 것"을 디테일하게 설명해야 정확한 결과가 나온다
- 원칙 3 : 환경을 충분히 묘사하라
  - 단순히 "주방에"가 아니라 "현대적인 한옥 스타일 주방, 원목 아일랜드 테이블, 격자 문양 창호지" 배경이 구체적일수록 자연스러 합성이 됨.
 
- 프롬프트
  - 여성의 의상을 하늘색 크롬 가디건 + 실버 미니 스커트+ 하트 버클 벨트 + 실버 롱부츠로 변경해줘. 그리고 반드시 그녀의 얼굴은 그대로 유지해.
 

## 냅킨 AI 서비스
- 텍스트만 입력해주면 자동으로 도식을 만드는 도구
  - 텍스트를 인포그래픽, 다이어그램, 플로우 차트 등으로 자동으로 변환해줌
- 기획한 복잡한 내용을 글로만 정리하면 AI가 알아서 도식으로 변경을 해줌
  - EX) 신규 사업 기획서를 워드로 작성했다로 가정할 때
    - 텍스트를 그대로 냅킨에다가 복사 붙여넣기만 하면 됨

- 프롬프트를 잘 쓰는 툴이 아니라 이미 작성한 텍스트를 복사 붙여넣기를 해서 시각화할 부분만 선택하여 ai가 알아서 최적의 도식을 제안해주는 것
