# [5차시] 완성도를 높여가는 기획안 작성
- 클로드 특징 설명
  - 작문 능력이 뛰어남, 기획서 작성이 뛰어남
- ai에게 기획서를 써달라고 막연하게 쓰면 그지 같이 나옴
- 기획서 구조 설명
  - 사실 -> 문제 -> 해결책
  - why-what-how-so what 구조화된 사고법
  - 배경-현황-추진과제-실행계획-기대효과
  - 제일 중요한 사내 정보 안전하게 활용하면서 이런 ai를 활용할 수 있는 데이터가 학습에 활용되지 않는 실무적인 보안 방
