# [5차시] 완성도를 높여가는 기획안 작성
- 클로드 특징 설명
  - 작문 능력이 뛰어남, 기획서 작성이 뛰어남
- ai에게 기획서를 써달라고 막연하게 쓰면 그지 같이 나옴
- 기획서 구조 설명
  - 사실 -> 문제 -> 해결책
  - why-what-how-so what 구조화된 사고법
  - 배경-현황-추진과제-실행계획-기대효과
  - 제일 중요한 사내 정보 안전하게 활용하면서 이런 ai를 활용할 수 있는 데이터가 학습에 활용되지 않는 실무적인 보안 방

## 실습 목표
- 예시 : 고객 응대팀을 위한 ai에이전트를 기획 가정

## 보안 정책 이해하기
- 기획 시 회사에 정보가 입력돼야 함
  - chatgpt: 옵트 아웃이라는 기능을 사용하여 별도로 꺼야 함
  - 클로드 : 개인정보가 학습되지 않음

## 기획서 첫단계 : why
- 현 상황을 클로드에게 전달, 문제점을 체계적으로 도출

~~~

도대체 왜 필요한지 기획의 첫 단계 "why"입니다.
현재 상황

- 문의 유형 500개(이미 분류 완료)
- 메뉴얼 확인 후 개별 응답 방식
구체적인 예시와 함께 설명해주세요.
~~~

## 기획서 두 번째 단계: What
- 구체적으로 무엇을 해결해야 하는지 어떤 결과를 달성해야 하는지 명확하게 정의하는 곳
- 앞서 파악한 문제들을 구체적인 해결과제로 변환하는 과정
  - 측정 가능한 목표 설정 필요

~~~
앞서 파악한 CS 운영 문제점들을 바탕으로 구체적인 해결과제를 도출하세요.
현재 문제점(가정) :
다음과 같이 해결과제를 구체화주세요:

2.정성적 목표:
3.핵심기능 요구사항:
각 목표에 대해 측정 가능한 지표와 달성 기준을 함께 제시해주세요.
~~~

## 기획서 세 번째 단계 : HOW
- 구체적으로 어떻게 목표를 달성할 것인지 => 실현 방안 설계
- 추상적인 아이디어를 구체적인 실행 계획으로 변환 => 기술적 실현 가능성과 비즈니스 제약 사항 모두 고려

~~~
ai 에이전트 도입을 통해 앞서 정의한 목표들을 달성하기 위한 구체적인 실행 방안을 설계해주세요.
~~~

~~~
앞서 제시한 실행 방안을 다음 예산 범위 내에서 최적화해주세요.
1. 시장 조사를 진행하고 난 다음에 현실적인 금액을 파악한다.
2. 각 실행 단계별 비용 대비 효과를 분석한다.
3. 제한된 예산 내에서 핵심 목표 달성을 위한 우선순위를 설정한다.
4. 단계별 실행 일정과 예상 소요 시간을 제시한다.
결과물로 다음을 포함해 주세요.
- 최적화된 실행 계획(우선순위별)
- 단계별 예상 비용 및 일정
- 예산 절약 방안 또는 대안책
~~~
